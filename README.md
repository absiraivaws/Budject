# Budject - Budget & Money Manager

A modern, responsive web-based budget and money management application built with React and Vite.

## Tech Stack

- React + Vite
- IndexedDB for local storage
- Vanilla CSS with design system
- Chart.js for visualizations
- React Router for navigation
